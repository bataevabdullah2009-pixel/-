# Roadmap

Текущий этап — production-oriented MVP голосового журнала продаж: стабильный voice pipeline, owner authentication, shop isolation, отчёт, журнал и ручная коррекция.

Ближайший эксплуатационный backlog:

- применить миграции в целевом Supabase и провести end-to-end smoke run;
- настроить резервное восстановление и мониторинг внешних интеграций;
- добавить CI для `npm run test` и `npm run build`.

CRM, склад, касса, оплаты и клиентская база не входят в roadmap текущего продукта.
