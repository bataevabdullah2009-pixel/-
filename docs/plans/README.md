# Планы

Планы разделены по состоянию, а каждая существенная задача имеет собственную папку и проверяемый результат.

| Раздел | Назначение |
| --- | --- |
| [Активные](./active/README.md) | Текущая работа: план, контрольный список, приёмка и заметки. |
| [Завершённые](./completed/README.md) | История реализованных этапов и доказательства проверки. |
| [Будущие](./backlog/README.md) | Идеи после MVP, не входящие в текущую сдачу. |

Жизненный цикл: создать план в `active`, выполнить контрольный список, зафиксировать проверку и перенести результат в новую папку `completed`. Завершённые номера отражают хронологию результата.
