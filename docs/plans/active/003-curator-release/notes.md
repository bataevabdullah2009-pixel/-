# Заметки

- Внешние проверки не заменяются моками.
- Webhook и polling являются взаимоисключающими режимами.
- После изменения окружения Vercel требуется повторное развёртывание.
- Новая миграция применяется до зависимого кода.
- Перед внешней проверкой применить `20260619132225_align_sale_item_update_and_exclusion.sql`; локальные проверки не подтверждают схему удалённого Supabase.
- Опасные тестовые действия выполняются только в демонстрационном магазине.
- Идемпотентность и транзакционная запись не входят в текущую реализацию и обязательны до публичного использования.
- Одновременно поддерживается только этот активный план; дальнейшие этапы хранятся в backlog.
