# План: релиз для куратора

## Цель

Подготовить воспроизводимый curator-ready MVP для демонстрации на Vercel и в Telegram. Это единственный активный план.

## Проблема

Локальные тесты и сборка не подтверждают удалённую миграцию, переменные Vercel, Telegram webhook и кнопку Mini App.

## Объём работ

- применить миграции Supabase;
- настроить окружение Vercel и повторное развёртывание;
- установить webhook с секретом;
- проверить голосовой сценарий, отчёт и кнопку Mini App;
- проверить правку, исключение, восстановление и сброс на тестовых данных;
- зафиксировать короткий демонстрационный сценарий.

## Затрагиваемые области

`docs/specs/technical/deployment-vercel.md`, `docs/roadmap/roadmap.md`, `CHANGELOG.md` и внешняя конфигурация.

## Ограничения

- не коммитить `.env.local` и секреты;
- не запускать polling одновременно с рабочим webhook;
- не сбрасывать реальные данные;
- не менять webhook без подтверждённой причины.

## Порядок

1. Проверить локальные команды.
2. Применить миграции и проверить схему.
3. Проверить окружение и повторное развёртывание.
4. Установить webhook и проверить его состояние.
5. Выполнить сквозную демонстрацию и записать результат.
