# Acceptance

Работа принимается, когда валидная voice sale появляется в отчёте своего магазина; чужой магазин недоступен; `needs_review`, `needs_price` и soft-deleted позиции не входят в итог; ручная правка и исключение работают через server actions; неизвестный продавец и неверный initData не создают доступ; `npm run test` и `npm run build` завершаются успешно.
