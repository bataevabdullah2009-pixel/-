# Активный план

Одновременно активен только один предметный план.

| План | Состояние | Результат |
| --- | --- | --- |
| [003 Curator release](./003-curator-release/plan.md) | Ожидает внешних проверок | Проверенный демонстрационный deployment, webhook и приёмочный сценарий. |

Перед изменением проекта прочитать [plan.md](./003-curator-release/plan.md), [checklist.md](./003-curator-release/checklist.md), [acceptance.md](./003-curator-release/acceptance.md) и [notes.md](./003-curator-release/notes.md).

Следующий production-этап нельзя добавлять сюда параллельно: сначала завершить или явно остановить текущий план, затем перенести один выбранный backlog-пункт.
