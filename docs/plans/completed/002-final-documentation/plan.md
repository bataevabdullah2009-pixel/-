- `README.md`, `AGENTS.md`, `docs/INDEX.md`;
- роли, production-readiness и семь технических спецификаций;
- правила, roadmap, планы и проектный Codex skill;
- удаление пустого legacy-каталога навыка и устаревших путей;
- проверка ссылок, пустых папок, тестов и сборки.

## Ограничения

Не менять приложение, Supabase migrations, webhook, STT, LLM parser, отчёты, UI и `.env.local`.
