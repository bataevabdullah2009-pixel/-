# Приёмка

- `docs/INDEX.md` ведёт ко всем разделам;
- нет ссылок на удалённые документы и прежний каталог навыка;
- в `docs/plans/active` находится ровно один предметный план;
- production-долги имеют текущий статус, риск, план и критерий;
- все каталоги `docs` содержат Markdown-файлы;
- `.env.local` не изменён и не отслеживается Git;
- `npm run lint`, `npm run test` и `npm run build` проходят.
