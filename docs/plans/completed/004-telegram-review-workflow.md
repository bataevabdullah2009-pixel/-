# 004 — Telegram WebApp review workflow

Закрыто: UI для корректировки товаров, отдельные блоки активных/review/excluded позиций, confirm action для legacy review cases, soft delete и restore.

Актуальный статус изменён планом 005: ручное сохранение валидных товара, количества и цены теперь сразу делает позицию «Готово», а новые уверенные voice-позиции не требуют обязательной проверки.
