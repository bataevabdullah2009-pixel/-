# 004 — сценарий проверки Telegram WebApp

Закрыто: UI для корректировки товаров, отдельные блоки активных/проверяемых/исключённых позиций, действие подтверждения для устаревших случаев проверки, мягкое удаление и восстановление.

Актуальный статус изменён планом 005: ручное сохранение валидных товара, количества и цены теперь сразу делает позицию «Готово», а новые уверенные голосовые позиции не требуют обязательной проверки.
