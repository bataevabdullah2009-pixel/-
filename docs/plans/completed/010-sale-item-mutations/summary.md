# Стабилизация мутаций товаров

19 июня 2026 года исправлены сохранение и мягкое исключение `sale_items` в Web App без изменения Telegram bot, webhook, STT или LLM-парсера.

Добавлена согласующая migration для `updated_at`, `excluded`, `excluded_by_owner` и индекса активных позиций. Серверные функции обновляют одну строку по ID, пересчитывают отчёт и возвращают безопасные пользовательские ошибки.
