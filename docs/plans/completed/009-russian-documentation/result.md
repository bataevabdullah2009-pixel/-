# Русская документация и правила GitHub — результат

Документация теперь описывает фактическую реализацию и честно отделяет учебный MVP от требований реальной эксплуатации. Исправлена неверная принадлежность полей мягкого удаления, зафиксированы отсутствие авторизации, идемпотентности, транзакций и CI, а также риски неявного демонстрационного режима, сброса без фильтра магазина и частичных записей.

Порядок работы с GitHub документирован без выполнения внешних действий: коммит, push и Pull Request не создавались.
