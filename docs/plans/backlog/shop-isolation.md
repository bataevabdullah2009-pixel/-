# Изоляция магазинов

## Цель

Гарантировать, что все чтения, мутации, отчёты и аудио ограничены `shop_id` из проверенной сессии.

## Результат

- обязательные серверные фильтры;
- deny-by-default RLS;
- scoped Storage policies;
- безопасный сброс дня;
- тесты A не видит/не меняет B.

Активировать после выбора механизма owner auth или в одном согласованном плане с ним.
