# Политика backup

## Цель

Определить и проверить резервирование Supabase Postgres и Storage.

## Результат

- согласованные RPO/RTO;
- расписание, retention, шифрование и доступ;
- отдельное покрытие аудио Storage;
- пошаговый restore;
- регулярный тест восстановления с протоколом.

Детали: `docs/specs/technical/backup-and-restore.md`.
