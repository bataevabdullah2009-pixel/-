# Backlog

Backlog содержит идеи после MVP. Эти задачи нельзя добавлять молча, потому что часть из них может расширить проект слишком сильно.

## Safe after MVP

| Task | Reason |
| --- | --- |
| Export to Excel | Помогает владельцу сохранить отчёт. |
| Better edit form | Улучшает исправление спорных позиций. |
| PWA mode | Помогает открыть отчёт с телефона. |
| Owner/seller roles | Делает доступ безопаснее. |
| More tests | Повышает стабильность. |

## Requires product decision

| Task | Risk |
| --- | --- |
| Product catalog UI | Может превратиться в склад. |
| Price history | Может превратиться в товароучёт. |
| Seller performance stats | Может превратиться в CRM/аналитику. |
| Categories | Может привести к сложной аналитике. |

## Not allowed now

- Остатки.
- Списание товара.
- Касса.
- Оплата.
- Фискализация.
- Клиентская база.
