# Обработка голоса

Бот принимает voice message только от активного продавца. Аудио скачивается, при необходимости конвертируется, отправляется в STT, затем очищается и разбирается LLM. Evidence layer повторно извлекает количество и цену из исходного STT-текста, чтобы модель не могла выдумать числа.

Нормальная фраза «Чипсы Принглс 20 штук по 300 рублей» создаёт `quantity = 20`, `price = 300`, `total = 6000`, `status = processed`.
