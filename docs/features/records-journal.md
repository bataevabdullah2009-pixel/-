# Журнал записей

Журнал показывает продажи магазина, продавца, исходный и очищенный текст, понятный русский статус, итог и временную signed URL аудио. `processed` отображается как «Подтверждено», review states — как «Нужно проверить»; internal enum пользователю не показывается. Поддерживаются период, продавец и текстовый поиск; seller filter не расширяет shop boundary.

Журнал является Server Component: он использует повторно валидированную HttpOnly initData cookie и всегда добавляет `owner.shopId` в Supabase query.
