# Журнал записей

Журнал показывает продажи магазина владельца, продавца, исходный и очищенный текст, статус, итог и временную signed URL аудио. Поддерживаются период, продавец и текстовый поиск; seller filter не расширяет shop boundary.

Журнал является Server Component: он использует повторно валидированную HttpOnly initData cookie и всегда добавляет `owner.shopId` в Supabase query.
