# Records Journal

Журнал показывает voice-продажи текущего server-derived магазина.

Карточка записи содержит:

- дату и время;
- продавца;
- распознанный или сырой текст;
- пользовательский статус;
- сумму;
- кнопку «Прослушать аудио», если audio сохранено;
- раскрытие «Товары».

Для `needs_review` показывается бейдж «Нужно проверить».

В журнале нет кнопок подтверждения или отмены записи.

Internal enum не показываются. Labels:

- `processed` -> «Готово»;
- `needs_review`, legacy `needs_price`, `failed`, `pending` -> «Нужно проверить»;
- `cancelled`, `excluded` -> «Исключено».

Поддерживаются период, продавец и текстовый поиск.

Seller filter не расширяет shop boundary.

Auth или Supabase error отображается отдельным сообщением и не превращается в «Записей нет».
