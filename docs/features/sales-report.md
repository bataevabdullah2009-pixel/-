# Sales Report

Отчёт показывает выручку и количество за выбранный период.

В итог входят только активные готовые позиции (`processed`; legacy `confirmed` также распознаётся расчётчиком). Review и excluded позиции не учитываются.

Review-позиции всегда выводятся отдельно в «Нужно проверить». Ошибка Telegram auth или Supabase не маскируется под пустой отчёт.

Периоды считаются по `Europe/Moscow` с полуинтервалом `[start, end)`.

На странице есть:

- summary bar с периодом, выручкой и количеством;
- таблица/карточки товаров;
- блок «Нужно проверить»;
- блок «Готовые продажи»;
- список исключённых товаров с восстановлением;
- действия refresh и reset day.
