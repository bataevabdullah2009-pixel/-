# Sales Report

Отчёт показывает выручку и количество за выбранный период.

В итог входят только активные `processed` позиции. Review и excluded позиции не учитываются.

Если данных нет, показывается empty state: «Нет активных продаж за выбранный период». Это не ошибка.

На странице есть:

- summary bar с периодом, выручкой и количеством;
- таблица/карточки товаров;
- блок «Нужно проверить»;
- блок «Готовые продажи»;
- список исключённых товаров с восстановлением;
- действия refresh и reset day.
