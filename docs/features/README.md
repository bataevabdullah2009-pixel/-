## MVP-фичи

- Telegram-бот для голосовых продаж.
- Vercel webhook для production updates от Telegram.
- Локальный polling для разработки.
- STT-распознавание голоса.
- LLM cleanup.
- LLM JSON parser.
- Supabase Storage для аудио.
- Supabase Postgres для записей, продаж и позиций.
- Простая таблица товаров с ценой по умолчанию.
- Веб-панель записей.
- Веб-панель продавцов.
- Дневной отчёт.
- Недельный, месячный и годовой режимы отчёта.
- Группировка одинаковых товаров.
- Блок “Нужно проверить”.
- Ручное исправление спорной позиции.
- Адаптивный интерфейс для Telegram Mini App с нижней навигацией и мобильными карточками.

## Главное ограничение

Фичи должны усиливать голосовой журнал продаж. Всё, что требует остатков, кассы, клиентской базы, оплаты, фискализации или полноценной CRM, не входит в текущий проект.
