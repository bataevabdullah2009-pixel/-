## Функции MVP

- Telegram-бот для голосовых продаж и Vercel webhook для рабочих update.
- Локальный polling для разработки.
- STT, очистка текста и строгий LLM-парсер.
- Supabase Storage для аудио и Postgres для записей, продаж и позиций.
- Цены товаров по умолчанию без складской логики.
- Страницы записей, продавцов и отчёта.
- Периоды день, неделя, месяц, год и выбранная дата.
- Группировка одинаковых товаров и блок «Нужно проверить».
- Правка, обратимое исключение, восстановление и сброс дня.
- Адаптивный интерфейс Telegram Mini App.

Функции должны усиливать голосовой журнал продаж. Остатки, касса, клиентская база, оплата, фискализация и полноценная CRM не входят в проект.
