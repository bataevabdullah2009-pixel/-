# Данные

Запрещено физически удалять `sale_items` из пользовательских действий. Активные отчётные запросы учитывают только `deleted_at is null`. Денежные итоги пересчитываются на сервере и включают только подтверждённые позиции. Изменение schema требует безопасной migration с сохранением существующих строк и синхронного обновления database docs; текущий review workflow использует существующие статусы и не требует новой migration.
