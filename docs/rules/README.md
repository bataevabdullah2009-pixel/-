## Непереговорные правила

- Проект остаётся простым голосовым журналом продаж.
- Нельзя превращать его в CRM, склад, кассу или товарный учёт.
- ИИ не придумывает товар, цену, количество, категорию или аналитику.
- Неизвестная цена получает `needs_price` и не входит в выручку.
- Основной отчёт показывает только товар, количество и выручку.
- Ошибки журналируются без секретов.
- Webhook проверяет `x-telegram-bot-api-secret-token`.
- Polling не запускается на Vercel.
- Изменение обновляет профильные документы, `CHANGELOG.md` и активный план.
- Вся проектная документация пишется на русском языке.
- Новые `README.md` и дублирующие документы не создаются.

## Проверка изменения

1. Изменение помогает пути «голос → текст → продажа → отчёт»?
2. Оно не превращает проект в другую бизнес-систему?
3. Оно описано в спецификациях, функциях и плане?
4. Оно проверено lint, тестами и сборкой?
5. Оно не раскрывает секреты и не обходит авторизацию?

Если обязательный ответ отрицательный, изменение нельзя считать готовым.
