# AI Rules

LLM извлекает только факты из STT-текста. Количество и цена подтверждаются deterministic evidence layer. Нельзя выдумывать цену, количество или товар.

`confidence >= 0.80` требуется для автоматического `processed`. Низкая уверенность, отсутствующая цена/количество, странный текст или parser fallback дают `needs_review`.

Внутренние enum не показываются пользователю.
