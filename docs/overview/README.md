# Обзор

Начните с [описания продукта](./product.md), затем прочитайте [глобальную спецификацию](../specs/global.md), [архитектуру](../specs/technical/architecture.md) и [модель данных](../specs/data/data-model.md).

Границы MVP: голосовая фиксация продаж, ручное исправление, исключение ошибочных позиций, журнал, продавцы и отчёт владельца. CRM, склад, касса, оплата и клиентская база не входят в продукт.
