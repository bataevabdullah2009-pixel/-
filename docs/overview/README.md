# Обзор

Voice Sales Log помогает небольшому магазину вести продажи голосом. Продавец отправляет сообщение в Telegram, система распознаёт товар, количество и цену, сохраняет продажу и показывает владельцу отчёт.

Mini App открывается двумя штатными способами:

- Telegram mode — через Telegram `web_app` кнопку с валидным `initData`.
- Browser fallback mode — напрямую в браузере, если `ALLOW_WEBAPP_FALLBACK=true` и сервер знает `DEFAULT_SHOP_ID` / `DEFAULT_SELLER_ID`.

Готовые позиции сразу попадают в отчёт. Проверка нужна только когда распознавание неполное или ненадёжное. Диагностика `/debug-telegram` показывает признаки SDK/initData, но не является обязательной для работы отчёта.

Границы MVP: голосовая фиксация продаж, отчёт, журнал записей, продавцы, ручная корректировка и soft delete. CRM, склад, касса, оплаты и клиентская база не входят в продукт.
