# Обзор продукта

`Голосовой журнал продаж` помогает магазину вести выручку через голосовые сообщения Telegram: продавец говорит продажу, бот распознаёт товары и цены, Supabase хранит запись, WebApp показывает отчёт.

Этот раздел оставлен как короткий обзор. Канонические правила продукта находятся в [../specs/global.md](../specs/global.md).

## Быстрый путь

1. Продавец отправляет голосовое сообщение в Telegram.
2. Бот распознаёт продажу и сохраняет позиции.
3. Уверенные позиции сразу попадают в отчёт.
4. Сомнительные позиции попадают в проверку.
5. Владелец работает с WebApp: `Отчёт`, `Проверка`, `Записи`, `Продавцы`.
