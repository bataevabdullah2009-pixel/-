# Mobile Workplan

## Цель

Финально адаптировать существующую веб-панель `voice-sales-log` для Telegram Mini App без изменений backend-логики, webhook, STT, LLM parser и Supabase schema.

## Выполнено

- [x] Desktop-таблица отчёта заменяется на карточки продаж на mobile.
- [x] Устранён горизонтальный скролл на узком экране.
- [x] Добавлен sticky summary с периодом, выручкой и количеством.
- [x] Фильтры “Сегодня / Вчера / Неделя / Месяц / Год” компактно переносятся.
- [x] Добавлена кнопка “Обновить”, повторно загружающая Server Component и данные Supabase.
- [x] Добавлена нижняя mobile-навигация; desktop-навигация сохранена сверху.
- [x] Карточки записей показывают статус, текст, сумму, аудио и исправление спорных записей.
- [x] Блок проверки получил заметные состояния для нулевого и ненулевого количества.
- [x] Добавлены empty states и loading skeleton.
- [x] Размеры заголовков, отступов, фильтров, таблиц и карточек уменьшены на mobile.

## Ограничения

- Бизнес-логика, Telegram webhook, STT, LLM parser и Supabase schema не менялись.
- Склад, CRM, оплата и касса не добавлялись.

- [x] `npm run lint`
- [x] `npm run test` — 25 тестов
- [x] `npm run build`
- [x] Визуальная проверка desktop и mobile viewport 390×844
