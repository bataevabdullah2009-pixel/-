# Features

## Готово в MVP

- Приём голосовых сообщений в Telegram-боте.
- Скачивание аудио из Telegram.
- Загрузка аудио в Supabase Storage.
- Распознавание голоса через Whisper-compatible STT API.
- Очистка текста через LLM.
- Извлечение товарных позиций через строгий JSON.
- Сохранение голосовых записей.
- Сохранение продаж.
- Сохранение позиций продажи.
- Поиск цены в `products`, если цена не названа голосом.
- Статус `needs_price` для позиций без цены.
- Страница записей `/records`.
- Страница продавцов `/sellers`.
- Страница отчёта `/daily-report`.
- Группировка одинаковых товаров.
- Итоговая сумма только по позициям с известной ценой.
- Блок “Нужно проверить”.
- Минимальное ручное исправление спорной позиции.

## Ограничения

- Нет складских остатков.
- Нет фискализации.
- Нет клиентской базы.
- Нет автоматической аналитики по категориям.
- Нет онлайн-оплаты.
- Нет полноценной авторизации владельца.

## Запланировано после сдачи MVP

- Экспорт отчёта в Excel.
- Более удобное редактирование спорных позиций.
- Роли владельца и продавца.
- PWA-режим.
