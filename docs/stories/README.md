# Каталог пользовательских историй

Истории разделены по ролям и проверочным сценариям.

| Документ | Назначение |
| --- | --- |
| [Истории продавца](./seller.md) | Голосовой ввод и обратная связь бота. |
| [Истории владельца](./owner.md) | Отчёт, поиск и управление позициями. |
| [Истории куратора](./teacher.md) | Проверка, документация и GitHub-процесс. |
| [Приёмочные сценарии](./acceptance-scenarios.md) | Наблюдаемые проверки продукта. |

Ключевой путь: продавец говорит товар, количество и цену; владелец получает отчёт, исправляет сомнительные данные и сохраняет исходный след.
