# Техническая архитектура

Каноническая архитектура проекта находится в [../../architecture/architecture.md](../../architecture/architecture.md).

Этот файл оставлен как совместимая ссылка из раздела технических спецификаций, чтобы старые ссылки не ломались и не возникало дублирования архитектурного описания.

## Как читать

1. Для устройства модулей, webhook, voice pipeline, WebApp, Supabase и границ безопасности читать [архитектуру](../../architecture/architecture.md).
2. Для бизнес-правил продукта, статусов, проверки, подтверждения, отмены и выручки читать [глобальную спецификацию](../global.md).
3. Для БД читать [database.md](./database.md) и [database-schema.md](./database-schema.md).
