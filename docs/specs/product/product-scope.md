# Назначение продукта

## Задача пользователя

Когда продавец завершает продажу, ему нужно за несколько секунд зафиксировать товар, количество и цену без кассовой или CRM-формы. Когда владелец проверяет день, ему нужен готовый итог и способ исправить распознавание.

## Полный сценарий

Продавец: открыть Telegram-бота → записать голос → отправить → получить подтверждение сохранения или понятную ошибку.

Владелец: открыть Menu Button → выбрать период → посмотреть карточки и сводку → при необходимости исправить, исключить или восстановить позицию. Для сброса итогов выбрать конкретный день и подтвердить действие.

Ценность: меньше ручного ввода для продавца, меньше ручного подсчёта для владельца и сохранённый исходный след для проверки.
