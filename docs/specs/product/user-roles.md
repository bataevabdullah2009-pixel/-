# Роли пользователей

## Продавец

Использует чат бота, `/start`, голосовое сообщение и статус. Не редактирует БД и не видит системные ошибки или секреты.

## Владелец

Использует Web App: итоги, сгруппированные товары, записи, продавцов, фильтры и корректирующие действия. Опасные действия подтверждаются и обратимы.

## Куратор

Проверяет воспроизводимость, архитектурные границы, качество документации и приёмочный сценарий. Ему не нужны рабочие секреты; `.env.example` показывает только имена переменных.

## Примечание об авторизации

Это продуктовые роли. Текущий демонстрационный проект не реализует надёжную техническую авторизацию владельца. До работы с реальными данными она обязательна.
