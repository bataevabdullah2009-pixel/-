# Owner Dashboard

Владелец видит:

- отчёт по выручке и количеству;
- блок «Нужно проверить»;
- список готовых товаров;
- исключённые товары с восстановлением;
- журнал записей;
- список продавцов.

Все чтения и изменения выполняются на сервере после `resolveRequestContext()` / `requireOwner()`. Seller shop имеет приоритет, чтобы совпадать с bot save. Сохранение валидных товара, количества и цены переводит позицию в «Готово» и сразу пересчитывает отчёт. Исключение и восстановление проверяют изменённую строку и обновляют UI через revalidation без ручной перезагрузки.
