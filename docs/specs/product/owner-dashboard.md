# Панель владельца

Панель содержит сводку количества и выручки, список активных позиций, блок ручного исправления, исключённые позиции, журнал записей и список продавцов.

Вход в панель начинается только из Telegram `web_app`: inline-кнопки и нижняя menu button используют один production URL. До успешной проверки initData защищённые данные не показываются; UI различает отсутствующий/невалидный initData, непривязанный Telegram и отсутствующий магазин.

Все чтения и изменения выполняются на сервере после `requireOwner()`. Фильтр магазина добавляется сервером. После изменения позиции маршруты `/daily-report` и `/records` инвалидируются.
