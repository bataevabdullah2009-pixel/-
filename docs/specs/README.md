# Спецификации

Спецификации описывают текущее проверяемое поведение `voice-sales-log`. Они не являются списком будущих идей или планом работ.

| Раздел | Содержание |
| --- | --- |
| [Глобальная спецификация](./global.md) | Проблема, решение, пользовательский путь и границы системы. |
| [Технические спецификации](./technical/README.md) | Стек, API, голос, БД, интерфейс, безопасность, тесты и развёртывание. |
| [Продуктовые спецификации](./product/README.md) | Границы, роли, MVP и критерии приёмки. |
| [Спецификации данных](./data/README.md) | Модель, статусы, аудит, удаление и восстановление. |

Готовой считается только функция, которая существует в коде или миграции и имеет способ проверки. Планируемые возможности размещаются в `docs/plans/backlog`.
