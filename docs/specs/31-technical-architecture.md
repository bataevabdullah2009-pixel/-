2. Для бизнес-правил продукта, статусов, проверки, подтверждения, отмены и выручки читать [глобальную спецификацию](./00-global.md).
3. Для БД читать [25-technical-database.md](./25-technical-database.md) и [26-technical-database-schema.md](./26-technical-database-schema.md).
