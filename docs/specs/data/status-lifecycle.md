# Жизненный цикл статусов

- `processed`: есть осмысленное название, количество, положительная цена, total и confidence не ниже 0.75.
- `needs_price`: товар и количество определены, цена отсутствует.
- `needs_review`: отсутствует количество, название неинформативно, confidence низкий или parser сообщил неоднозначность.
- `failed`: обработка завершилась ошибкой.
- `excluded`: владелец исключил позицию либо выполнен reset дня.

Ручное сохранение устанавливает `processed`, `confidence = 1`, `total = quantity × price` и новый `updated_at`.
